Typescript Template
=========

```
npm install
```

* typescript
* jest
* eslint
* prettier


## Scripts

| script |  description  |
|--------|---------------|
| `test`   | run test |
| `test:watch`   | run test watch |
| `dev`   | node typescript |
| `lint`  | linter |
| `build`  | build |
| `publish:lib`  | running all script then publish in npm. Remember to login first and assign a name and version in the package.|
